# sistema escolar

este projeto foi criado para ajudar na organização da escola, facilitando o controle de alunos, professores, frequência, notas e atividades escolares.

## problema

muitas escolas ainda organizam informações manualmente, o que pode causar erros, perda de dados e dificuldade no gerenciamento.

## objetivo

o objetivo do sistema é deixar a organização da escola mais prática e rápida, ajudando no controle das informações do dia a dia.

## funcionalidades iniciais

- login de usuários
- cadastro de alunos
- cadastro de professores
- controle de frequência
- cadastro de atividades
- lançamento de notas
- relatórios

## tecnologias utilizadas

- html
- css
- javascript
- node.js
- mysql
